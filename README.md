# Simple API
A simple api for sending and retrieving message. 

## Installation
Make sure you have following software installed in your system
* go 1.10.4
* dep 

### Clone repository and dependency
* `git clone https://github.com/Rikadewi/simple-api.git`
* `dep ensure`
### Run on local server
* `go run main.go`
### Access
* port: `localhost:8000`
